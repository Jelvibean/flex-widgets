The secret behind sibiling divs growing eveningly.

In the days when we use to only have Table structures, we had no issues with taking two cells in a row and having them growing eveningly together. When we started to use more css for design and started to frown upon tables for any structure design problem, in came the dilemma of how to have two sibling divs growing eveningly.

In the world of floating DIVs this was a task to get this to happen. Many times we resorted back to table when we needed this type of structure.

Luckly today in the world of flex we are able to create a parent that holds children that can be given the same rule and those containers can have children that will adhere to the changes registered by the parents.

Notice how no matter how much content is put into this container, your sibling "Child 1 Right" also grows or shrinks adhereing to it sibling on the left.

It also forces "Child 2 Left" and "Child 2 Right" to eveningly move down accordingly.
